## 시작

### React 세팅

1. node -v, npm -v
2. npm install npx -g
3. npx create-react-app tell-me-what-u-want
4. npm install prop-types
   > validate package
5. npm install axios
6. npm install @material-ui/core
  npm install @material-ui/icons
7. npm install react-router-dom
8. npm install react-naver-maps
